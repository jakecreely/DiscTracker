Frequently Asked Questions

To find the CEX ID for your item, please follow these steps:

  1. Visit the CEX website.
  2. Search for your item using the search bar on the homepage.
  3. Once you find the item, click on it to open the product page.
  4. Look at the URL of the page, and you'll see an ID number at the end of the URL (e.g., https://uk.webuy.com/product-detail?id=1234567 in this example the ID is 1234567).
  5. Copy the ID from the URL and paste it into input in the 'My Collection' page and add the item.

If you have any questions, feel free to contact us!
